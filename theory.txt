Multer ---> Multer is a node.js middleware for handling multipart/form-data, which is primarily used for uploading files.
it is used to read files like image , video , audio , etc in backend as backend can only read json format
means multer sirf multipart data ko read kary ga
it can store data in two ways:
1.disk storage --- means backend mein folder banao pic pehly wahah store hogi phir database mein 
2.memory storage --- means image ya any file ko buffer (numbers letters) mein read kary ga or phir cloudinary mein save kary ga , it use CPU
